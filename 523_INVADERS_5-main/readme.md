# Suraksha Kavach - Chrome Extension

Suraksha Kavach is a Chrome extension aimed at mitigating online frauds using Whois information and a Machine Learning (ML) model API.

## Download Extension Files

Before using the extension, download the necessary files from the provided link included in the file named `drivelinkforclassifier5_joblibanddataset`.

## Python Requirements

Make sure you have the following Python modules installed:

```bash
pip install google scikit-learn numpy pandas Flask Flask-Cors requests

## Features

- **Fraud Mitigation:** Utilizes Whois information and a Machine Learning model API to help identify and mitigate online frauds.

## Installation

Follow these steps to install the Suraksha Kavach Chrome extension:

1. Download the extension files from the provided link in the `drivelinkforclassifier5_joblibanddataset` file.

2. Open Google Chrome and go to `chrome://extensions/`.

3. Enable "Developer mode" in the top-right corner.

4. Click "Load unpacked" and select the folder containing the downloaded extension files.

5. The Suraksha Kavach extension should now appear in your Chrome extensions.

## Usage

1. **Activate Extension:**
   - Click on the Suraksha Kavach icon in your Chrome toolbar to activate the extension.

2. **Fraud Mitigation:**
   - As you browse websites, the extension will use Whois information and the ML model API to identify potential online frauds.
